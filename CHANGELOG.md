# Changelog

## 0.1.0 - 2026-05-05

Initial private MVP release.

### Added

- Local issue-train workflow from GitHub Issue intake through PR, CI, auto-merge, and cleanup.
- Claude CLI read-only phases for planning, review, and supervision.
- Codex CLI workspace-write phases for implementation and fixes.
- Prompt-contract validation, runner timeout handling, API key guards, and child-process environment allowlists.
- `autokit init`, `add`, `doctor`, `run`, `resume`, `retry`, `cleanup`, `list`, and `status` CLI commands.
- Fixture provisioning, unprotected full integration smoke, protected auto-merge safety gate, and release verification preflight evidence.
- Private tarball and checkout-link distribution paths.

### Not Included

- Registry publishing.
- General-purpose repository compatibility beyond the v0.1.0 support matrix.
- Paid-risk-gated SDK runner matrices.
