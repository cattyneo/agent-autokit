# autokit-implement

Use this skill for `implement` and `fix` phases.

## Rules
- Read the issue, plan, SSOT, routes, components, and tests before editing.
- Prefer established Next.js and shadcn/ui conventions over custom primitives.
- Add focused validation for UI behavior, accessibility, responsive layout, and data boundaries when touched.
- Never expose secrets, read provider auth files, or run live provider calls without explicit approval.
- Return the configured prompt contract YAML with changed files, validation, and residual risks.
